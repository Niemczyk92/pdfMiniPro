Quick guide

  1. Open a PDF
  2. Add Text: click on PDF, type. Enter finishes; Shift+Enter for a new line. Select to format (Bold, Italic, Underline, color, background)
  3. Image: insert from file or Ctrl+V. Drag corners to resize, top handle to rotate
  4. Draw: free-hand drawing with mouse, finger, or pen — adjust color & stroke width
  5. Shapes: rectangle, ellipse, line, arrow — click and drag to draw
  6. Fields: insert signature, date, address, or business card with one click
  7. Drag to move; double-click text to re-edit; Delete to remove
  8. Save or Print when done. Zoom with Ctrl+scroll or the controls in the bottom-right

Compress PDF

Compression rasterizes each page into a JPEG image at the chosen quality. Vector text becomes part of the image, but file size shrinks significantly.

🔒 Password protect

Encrypt the current PDF with a password. The output is a .pdfenc file (AES-256-GCM, 250k PBKDF2 rounds). Open it back via the Open button — drop the .pdfenc file or pick it normally.
⚠ No recovery: without the password, the file cannot be decrypted. The encryption is fully client-side — your password never leaves the browser.

🔓 Decrypt PDF

This file is encrypted. Enter the password to open it.

🧼 Sanitize document

Strip everything that could leak data: PDF metadata, hidden objects, JavaScript actions, and old incremental revisions. Annotations are baked into the visual layer first so nothing is "secretly editable" afterwards.

🛡 RegEx redact

Permanently remove sensitive data. Each match in the PDF text layer is covered by an opaque rectangle and the text is dropped from the underlying content. Save the document to bake these in.

⇄ Compare two PDFs

Both PDFs are rendered side by side. Red = removed in the original, green = added in the updated version. Unchanged text stays untouched.

{ } Fill form from JSON

Paste a JSON object and we'll match its keys to the PDF's form-field names (case-insensitive, substring match). Or paste an array to produce multiple filled copies as a ZIP.

Split PDF

Split the current PDF into multiple files. Each chunk downloads as its own PDF.
Each range becomes one output PDF. Use commas to separate; dashes for inclusive ranges.
e.g. 2 → produces files of pages 1-2, 3-4, 5-6, …

🔐 Sign PDF with X.509 certificate

Real PAdES-B-B signature embedded in the PDF. Acrobat Reader will display the signature widget and validate the certificate chain against installed trust roots.

Settings

Automatic local backup
Your profile, signatures, custom stamps, templates, and preferences are automatically snapshotted to your browser's IndexedDB. The snapshot survives page reloads — and is offered for restore after a new version of the app is installed if your localStorage was wiped.
Checking last backup…
Export / Import as file
Move your settings between devices, or keep an off-device backup. The exported JSON contains profile, signatures, stamps, templates, AI config, and UI preferences. No PDFs are included.
Reset
Wipe local data (profile, signatures, custom stamps, templates, recent colours, AI config). PDFs in the auto-save draft are not affected.
Local AI server (OpenAI-compatible)
Point this to a local LLM that exposes an OpenAI-compatible /v1/chat/completions endpoint — Ollama, llama.cpp server, LM Studio, vLLM, etc. When connected, AI features (Summarize PDF, Translate, Explain paragraph, Fill-in suggestions) appear in the toolbar. Leave blank to disable.
Not configured.
Privacy
Requests are sent directly from your browser to the URL you configured. Nothing is sent to Anthropic, OpenAI, or any third party by this app. For LAN servers, ensure CORS is enabled on your AI host (most LLM servers expose this via a config flag).
Local error log
JavaScript errors are captured to this log so you can attach them to a bug report. Logs never leave your device unless you copy or email them yourself.
No errors logged.
Report a problem / send feedback
Compose a bug report or feedback message. It opens in your default mail client (mailto:) with the error log attached — or you can copy the body to the clipboard.
Note: SMTP from a browser requires a server bridge. The mail-client option works on every device without any backend. If you want fully automatic SMTP delivery, add a small relay (e.g., a Cloudflare Worker or a small Node server) and we'll wire it in here.
